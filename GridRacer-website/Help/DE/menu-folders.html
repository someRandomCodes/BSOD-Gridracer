Menü „Ordner“

Root-Verzeichnis wählen
Root bedeutet „Wurzel“ und bezeichnet den Basis-Ordner, in dem Sie die Dateien ablegen wollen, die vom Internet aus angefragt werden können.
Achtung: Wählen Sie als Root-Verzeichnis keinen Ordner aus, der private Unterlagen enthält, die nicht für die Öffentlichkeit bestimmt sind.

Root mit dem Dateimanager anzeigen
Diese Funktion ruft das Root-Verzeichnis mit dem Windows-Explorer auf. Sie können so auf gewohnte und bequeme Weise Ihre Dateien in den Root-Ordner kopieren oder verwalten.

Root mit dem Browser abrufen
Dies startet Ihren Browser (z.B. Internet Explorer, Firefox, Opera, etc.) mit der Adresse "localhost", wodurch Sie Ihre eigenen Inhalte von diesem Webserver abrufen. Sie können so kontrollieren, wie Besucher Ihre Website bzw. Dateien sehen.

geschützte Verzeichnisse und Passworte
Hiermit rufen Sie das „Freunde/Mitglieder“-Fenster auf, in dem Sie passwortgeschützte Ordner und Benutzer verwalten können.
